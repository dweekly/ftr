Installation

macOS (Homebrew)

brew tap dweekly/ftr && brew install ftr

Linux (APT)

curl -sSL https://apt.networkweather.com/install.sh | sudo bash
sudo apt install ftr

Rust/Cargo

cargo install ftr

From Source

git clone https://github.com/dweekly/ftr
cd ftr
cargo build --release
